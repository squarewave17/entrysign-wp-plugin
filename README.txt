== Installation ==


1. Upload `entrysign-configurator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a page called 'app-test' In the wordpress site
4. Insert shortcode [shortcode_test] on the app-test page
